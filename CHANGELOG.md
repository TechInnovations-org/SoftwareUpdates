README - WebApp Project

Project Name: WebApp
Description:
This project is a sample web application demonstrating containerized deployment using Docker Compose.

Contents:
- docker-compose.yml : Defines services and containers for the application.
- CHANGELOG.md        : Lists all notable updates and version changes.
- src/                : Contains the main source code of the web application.
- docs/               : Project documentation and setup instructions.

Setup Instructions:
1. Clone the repository:
   git clone https://github.com/<your-username>/WebApp.git

2. Navigate to the project directory:
   cd WebApp

3. Start the application using Docker Compose:
   docker-compose up --build

4. Access the app at:
   http://localhost:3000
